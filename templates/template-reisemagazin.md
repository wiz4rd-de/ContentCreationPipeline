# Content-Briefing Vorlage: Reisemagazin

Dies ist das Template für jedes neue Reisemagazin-Briefing. Die Platzhalter in [eckigen Klammern] werden für jeden konkreten Artikel individuell befüllt.

---

## A. Meta-Daten & Steuerung

| Feld | Vorgabe |
|------|---------|
| **Arbeitstitel** | [z.B. „Die 10 schönsten Wasserfälle auf Island"] |
| **URL-Slug** | /reisemagazin/[keyword-slug] |
| **Haupt-Keyword** | [primäres Keyword inkl. Suchvolumen] |
| **Neben-Keywords** | [3–5 sekundäre Keywords / Long-Tails] |
| **Suchintention** | [informational / transactional-inspirational] |
| **Content-Format** | [Listicle (Variante A) / Thematischer Guide (Variante B)] |
| **Ziel-Wortanzahl** | [1.500–2.500 Wörter] |
| **Anzahl Sektionen** | [5–10 H2-Abschnitte] |
| **Anzahl Bilder** | [5–10, inkl. Hero-Bild] |
| **Zielgruppe** | [z.B. Familien, Abenteurer, Kulturreisende] |
| **Wettbewerber-URLs** | [2–3 URLs, die aktuell ranken] |
| **Geplante Interne Links** | [Liste relevanter /urlaub/- und /reisemagazin/-Seiten] |

## B. Headline & Intro

### H1 – Hauptüberschrift

| Aspekt | Vorgabe |
|--------|---------|
| **Muster** | [Destination/Thema] – [Emotionaler Hook / Versprechen] |
| **Länge** | Max. 65 Zeichen (für Google SERP) |
| **Keyword-Platzierung** | Haupt-Keyword am Anfang oder prominent eingebaut |
| **Beispiele (Live)** | „Island – Wasserfall-Wunder zwischen Feuer und Eis" / „Ayutthaya – Thailands vergessene Königsstadt" |
| **Ton** | Inspirierend, neugierig machend, nicht reißerisch |

### H2 – Untertitel (Subheadline)

| Aspekt | Vorgabe |
|--------|---------|
| **Muster** | [Konkretisierung des Versprechens / Teaser auf den Inhalt] |
| **Länge** | 1 Satz, max. 100 Zeichen |
| **Beispiele (Live)** | „Vom berühmten Skógafoss bis hin zu einem Geheimtipp" / „Geschichte zum Anfassen und das echte Thailand abseits des Trubels" |

### Intro-Absatz

| Aspekt | Vorgabe |
|--------|---------|
| **Länge** | 3–5 Sätze |
| **Aufbau** | 1) Szene setzen / Emotion wecken → 2) Thema einordnen → 3) Inhalt anteasern → 4) CTA mit DERTOUR-Einbindung |
| **Pflicht-Element** | Letzter Satz enthält DERTOUR-Verweis (z.B. „Entdecke ... mit DERTOUR") |
| **Keyword** | Haupt-Keyword mind. 1x im Intro enthalten |

---

## C. Content-Sektionen (Hauptteil)

Jede Sektion folgt einem einheitlichen Baukasten-Prinzip:

| Baustein | Vorgabe |
|----------|---------|
| **H2-Überschrift** | Listicle: „[Nr.] [Name/Ort]: [Beschreibendes Attribut]" · Guide: „[Thema/Aspekt der Destination]" · Keyword mind. in jedem 2. H2 enthalten. |
| **Teaser-Subtitle** | 1 kurzer Satz (ca. 50–80 Zeichen), der Lust auf die Sektion macht. Wird als Akkordeon-Label und mobile Vorschau genutzt. |
| **Lead-Absatz** | 1–2 Sätze, die den Kern der Sektion zusammenfassen. Enthält wenn möglich eine überraschende Zahl, ein Superlativ oder eine Anekdote. |
| **Bild** | 1 hochwertiges Foto mit: • Beschreibendem Alt-Text (inkl. Keyword, max. 125 Zeichen) • Bildunterschrift (emotionaler kurzer Satz) |
| **Fließtext** | 2–4 Absätze (je 50–100 Wörter). Mix aus: • Fakten (Zahlen, Maße, Entfernungen) • Storytelling (Geschichte, Legende, persönliches Erleben) • Praktischen Tipps (Anreise, Öffnungszeiten, Preise) • Interne Links (kontextuell, nicht forciert) |
| **Optional: Info-Box** | „Insider-Tipp" / „DERTOUR Extratipp" / Faktenbox mit kompaktem Spezialwissen |
| **Optional: Hotel-Teaser** | Eingebetteter Produktteaser mit Hotel-Name, Region, Bewertung, Preis ab X € |

---

## D. Sonder-Elemente (nach Bedarf)

| Element | Einsatz |
|---------|---------|
| **FAQ-Sektion** | 3–5 häufige Fragen zum Thema als ausklappbare Akkordeons am Artikelende. Ideal für Featured Snippets. |
| **Karten-Embed** | Google Maps Embed für Orte/Routen, wenn die Destination einen klaren geografischen Fokus hat. |
| **Bildergalerie** | Horizontale Scroll-Galerie bei vielen visuellen Highlights (z.B. Portrait-Darstellungen einzelner Sehenswürdigkeiten). |
| **Faktbox / Infokasten** | Hervorgehobener Kasten für komprimierte Fakten (z.B. Adresse, Öffnungszeiten, Eintrittspreise). |
| **Produkt-Teaser** | Eingebettete DERTOUR-Angebote (Hotels, Rundreisen) mit Bild, Bewertung und Preis. |

---

## E. Abschluss-Bereich

| Element | Vorgabe |
|---------|---------|
| **Schluss-Absatz** | Zusammenfassung der Highlights + emotionaler Abschluss. Ggf. Verweis auf weiterführende Artikel. |
| **Related Content** | 3–4 Teaser-Karten zu thematisch verwandten Reisemagazin-Artikeln mit Bild, Titel und Kurzbeschreibung. |
| **CTA-Bereich** | Optional: Buchungs-CTA oder Newsletter-Verweis. |

---

## F. Output-Format

Non-Content-Elemente in Drafts werden als Blockquote-Marker dargestellt:

| Element | Format |
|---------|--------|
| **Bild (pro H2)** | `> **[Bild]** {Motiv} \| Alt: {Alt-Text}` |
| **Bildunterschrift (pro Bild)** | `> **[Bildunterschrift]** {Caption}` |
| **Info-Box** | `> **[CMS: Info-Box]** Insider-Tipp / DERTOUR Extratipp / Faktenbox` |
| **Hotel-Teaser** | `> **[CMS: Hotel-Teaser]** Hotel-Name, Region, Bewertung, Preis ab X €` |
| **Karten-Embed** | `> **[CMS: Karten-Embed]** Google Maps für Orte/Routen` |
| **Bildergalerie** | `> **[CMS: Bildergalerie]** Horizontale Scroll-Galerie` |
| **Related Content** | `> **[CMS: Related Content]** 3–4 Teaser-Karten zu verwandten Artikeln` |
| **CTA-Bereich** | `> **[CMS: CTA]** Buchungs-CTA oder Newsletter-Verweis` |

## G. SEO-Checkliste

- [ ] Haupt-Keyword in H1, mind. 2–3 H2s und Intro-Absatz enthalten
- [ ] Title Tag: [Keyword] + emotionaler Zusatz | DERTOUR (max. 60 Zeichen)
- [ ] Meta Description: Handlungsaufforderung + Keyword (max. 155 Zeichen)
- [ ] Alt-Texte aller Bilder sind beschreibend und enthalten das Keyword
- [ ] Mind. 3 interne Links zu verwandten /reisemagazin/-Artikeln
- [ ] Mind. 2 interne Links zu transaktionalen /urlaub/-Seiten
- [ ] FAQ-Sektion vorhanden (falls sinnvoll für das Thema)
- [ ] Keine Keyword-Kannibalisierung mit bestehenden Artikeln geprüft
- [ ] Breadcrumb korrekt: Reisemagazin > [Thema]
- [ ] Alle Bilder haben Bildunterschriften (Captions)

## H. Tonalitäts- und Stil-Guidelines

| Kategorie | So ja ✓ | So nicht ✗ |
|-----------|---------|------------|
| **Ansprache** | „Entdecke die schönsten Fjorde" | „Der Tourist sollte die Fjorde besuchen" |
| **Emotion** | „Tosende Wassermassen stürzen 44 Meter in die Tiefe" | „Der Wasserfall ist 44 Meter hoch" |
| **Storytelling** | „Einst soll ein Wikinger seinen Schatz hier versteckt haben" | „Es gibt eine Legende über einen Wikinger" |
| **Fakten-Einbindung** | „Der Eintritt kostet ab 20 €, die Fahrt dauert 30 Min." | „Die genauen Preise findest du auf der Website" |
| **Marke DERTOUR** | „Entdecke Islands Wasserfälle mit DERTOUR" | „Buche jetzt bei DERTOUR!" |
| **Interne Links** | Natürlich im Fließtext eingebettet | Aufgelistet am Seitenende ohne Kontext |
