# Content-Briefing Blaupause: Urlaubsseiten

Dies ist das Template für jedes neue Urlaubsseiten-Briefing. Die Platzhalter in [eckigen Klammern] werden für jede konkrete Destination individuell befüllt.

**Content-Typ-Definition:** Urlaubsseiten liefern einen transaktionalen und inspirierenden Gesamtüberblick über ein Urlaubsziel. Sie ordnen die Destination für die jeweilige(n) Zielgruppe(n) ein und kombinieren relevante Informationen mit SEO- und AIO-Optimierung.

---

## A. Meta-Daten & Steuerung

| Feld | Vorgabe |
|------|---------|
| **Arbeitstitel** | [z. B. „Mallorca-Urlaub"] |
| **URL-Slug** | /urlaub/[destination-slug] |
| **Haupt-Keyword (HKW)** | [Urlaub + Destination / Destination + Urlaub, inkl. Suchvolumen] |
| **Neben-Keywords** | [3–5 sekundäre Keywords / Long-Tails, inkl. Suchvolumen] |
| **Suchintention** | transaktional-inspirational |
| **Ziel-Zeichenanzahl** | 11.000–15.000 Zeichen (gesamt) |
| **Anzahl Bilder** | 1 Keyvisual + 1 Infobox-Bild + Bilder für 2 Image Walls |
| **Zielgruppen** | Familien (29–49 J.), Paare jung (20–39 J.), Paare älter (40+), [1–2 individuelle Zielgruppen nach Recherche] |
| **Wettbewerber-URLs** | [2–3 URLs, die aktuell ranken] |
| **Geplante interne Links** | [Liste relevanter /urlaub/- und /reisemagazin/-Seiten + Facettenseiten] |
| **Linkziel-Tabelle** | [Wird durch DERTOUR SEO-Team bereitgestellt] |

## B. Meta-Daten (Title & Description)

Längentest via: https://app.sistrix.com/de/serp-snippet-generator

### Meta-Title

| Aspekt | Vorgabe |
|--------|---------|
| **Aufbau** | HKW in unveränderter Form ganz am Anfang + ergänzender Zusatz |
| **Suffix** | `| DERTOUR` wird automatisch ergänzt – **nicht** mitliefern |
| **Keyword-Regeln** | Keine Stopwords, keine Satzzeichen, keine Flexion im HKW |
| **Sonderzeichen** | Nicht erlaubt |
| **Beispiel** | Mallorca-Urlaub – Sonne, Strände & Kultur entdecken |

### Meta-Description

| Aspekt | Vorgabe |
|--------|---------|
| **Länge** | Max. 155 Zeichen |
| **Keyword** | HKW enthalten, Stopwords erlaubt (zur grammatikalischen Korrektheit) |
| **Inhalt** | Klarer Ausblick auf den Seiteninhalt |
| **CTA** | Optional |

## C. Hauptüberschrift, Subline & Keyvisual

### H1 – Hauptüberschrift

| Aspekt | Vorgabe |
|--------|---------|
| **Länge** | Max. 50 Zeichen, nicht länger als zweizeilig auf der Live-Seite |
| **Keyword** | HKW enthalten; Bindestrich im Keyword erlaubt (z. B. Mallorca-Urlaub), Doppelpunkte erst **nach** dem HKW erlaubt, keine Stopwords im HKW |
| **Inhalt & Stil** | Spezifisch auf die Destination abgestimmt, nicht austauschbar. Neugierig machend, catchy, Wortspiele erlaubt – Sinn darf nicht verloren gehen |
| **Nicht erwünscht** | Austauschbare Phrasen wie „Urlaub in XY – Sonne genießen"; Dreier-Konstellationen wie „Sonne, Strand und Meer" |

### Subline

| Aspekt | Vorgabe |
|--------|---------|
| **Länge** | Max. 70 Zeichen |
| **Funktion** | Ergänzt die H1 um Gefühl, spiegelt ein Bedürfnis und zeigt Mehrwert auf |
| **Keyword** | Nicht zwingend, aber vorteilhaft |

### Keyvisual

| Aspekt | Vorgabe |
|--------|---------|
| **Text-Bild-Konsistenz** | Bild muss zur H1 und Subline passen – keine Text-Bild-Schere |
| **Alt-Text** | Beschreibend, max. 125 Zeichen, mit Keyword |
| **Qualität** | Prüfen, ob aktuelles Bild verwendbar ist; ggf. neues Bild nach Bildguidelines auswählen |

## D. Gute Gründe

| Aspekt | Vorgabe |
|--------|---------|
| **Überschrift** | Fester Aufbau: „Warum solltest du Urlaub in [Destination] machen?" |
| **Format** | 3 Bulletpoints |
| **Länge** | Max. 360 Zeichen gesamt, pro Bullet max. 120 Zeichen, max. 3–4-zeilig auf Desktop |
| **Gleichmäßigkeit** | Alle drei Punkte müssen auf der Live-Seite gleich lang sein |
| **Inhalt** | Was macht die Destination als Reiseziel besonders? Besondere Attraktionen, Strände, Nationalparks, Natur, Kulinarik, prädestinierte Aktivitäten – muss zur Destination passen |
| **Wichtig** | Gründe müssen sich klar voneinander unterscheiden. Gründe müssen später im Text erneut aufgegriffen werden. Keine allgemeinen Infos zu Reisezeit/Wetter/Klima (Ausnahme: echtes USP wie Klima der Kanaren) |

## E. Intro

| Aspekt | Vorgabe |
|--------|---------|
| **Zeichenanzahl** | 300–500 Zeichen |
| **Keyword** | HKW so weit vorn wie möglich |
| **Stil** | Dynamisch, knackig, aufmerksamkeitsstark – muss neugierig machen |
| **Inhalt** | 1) Destination repräsentativ einordnen → 2) Zielgruppenbezug: Für wen geeignet? → 3) Allgemeine Beschreibung & Highlights → 4) Zusammenfassung der Seiteninhalte |
| **Pflicht-Element** | Letzter Satz: textlicher CTA mit Aufforderung zum Buchen |

## F. Content-Sektionen (Hauptteil)

Jede Sektion folgt einem einheitlichen Baukasten-Prinzip. Alle H2-Überschriften max. 50 Zeichen, optionale Subline max. 70 Zeichen, mit Keyword.

---

### F.3 Beliebte Sehenswürdigkeiten

| Baustein | Vorgabe |
|----------|---------|
| **H2 + Subline** | Mit Keyword, max. 50 / 70 Zeichen |
| **Intro + Aufzählung** | Kurzer Einleitungstext, dann Aufzählung |
| **Themenfelder** | Historische & kulturelle Orte, Architektur & Bauwerke, Natur & Landschaften, Kunst & Kultur, Besonderes (Märkte, Erlebniswelten), Geheimtipps (falls vorhanden). Bei Non-Beach-Destinationen: Strände hier integrieren |

### F.4 Optionales Thema (nach Relevanz)

| Baustein | Vorgabe |
|----------|---------|
| **Bedingung** | Nur einfügen bei mittlerer Relevanz – Themen, die relevant für die Destination sind, aber keine eigene Seite rechtfertigen |
| **H2 + Subline** | Mit Keyword, max. 50 / 70 Zeichen |
| **Inhalt** | Fließtext zum optionalen Thema |

### F.5 Zielgruppenabschnitte

| Baustein | Vorgabe |
|----------|---------|
| **H2 + Subline** | Pro Zielgruppe eine eigene Sektion, mit Keyword, max. 50 / 70 Zeichen |
| **Feste Zielgruppen** | Familien mit jüngeren Kindern (0–5 J.), Familien mit älteren Kindern (6–12 J.), Paare jung (20–39 J.), Paare älter (40+) |
| **Individuelle Zielgruppen** | Max. 1–2 zusätzliche, selbst recherchiert (z. B. Strandurlauber, Kulinarikurlauber) |
| **Verlinkungen** | Aus Zielgruppenabschnitt auf Facettenseiten verlinken (Linkziele über DERTOUR SEO-Team) |

| **Autor** | Immer angeben mit „Veröffentlicht von [Name]" |

**Personalisierung (nicht im Index):**
- Familien mit jüngeren Kindern
- Familien mit älteren Kindern
- *(Perspektivisch: Paare Aktiv, Paare Erholung – aktuell ausgeklammert)*

---

### F.6 Klima

| Baustein | Vorgabe |
|----------|---------|
| **H2 + Subline** | Mit Keyword, max. 50 / 70 Zeichen |
| **Einführungstext** | Kurzer, faktenbasierter Einstieg als Direktantwort, max. 500–600 Zeichen. Vertiefung darf nach der Direktantwort folgen |
| **Inhalt** | Allgemeine Infos (Haupt-/Nebensaison, Temperaturen, Besonderheiten wie Regenzeit). Auf Ort eingehen, Unterregionen nur grob anschneiden – lieber quer- und tieferverlinken. Spezifische Themen (beste Reisezeit für Wandern, Familien, Paare etc.) nur kurz anschneiden, da Verlinkungen gesetzt werden |
| **Wenn „Beste Reisezeit"-Seite vorhanden** | Kurz halten, nur das Wichtigste, dann direkt verlinken. Duplicate Content vermeiden! **Keine Tabelle** einfügen |
| **Wenn keine „Beste Reisezeit"-Seite vorhanden** | Klimatabelle einfügen |
| **Infokasten** | „Klima in [Destination] auf einen Blick": Headline max. 30–50 Zeichen, immer 3 Bulletpoints, max. 80–100 Zeichen pro Bullet |

## G. FAQ-Sektion

| Aspekt | Vorgabe |
|--------|---------|
| **Anzahl** | Mind. 3, max. 7 Fragen |
| **Fragetypen** | Google-W-Fragen (Wer, Wie, Was etc.) – relevant für AIO. Auch Nicht-W-Fragen erlaubt (z. B. „Ist der Strand kinderfreundlich?") |
| **Fragenübernahme** | Fragen 1:1 aus der Recherche übernehmen, nur grammatikalische Fehler korrigieren |
| **Zeichenlänge Fragen** | Keine maximale Zeichenlänge |
| **Zeichenlänge Antworten** | 150–250 Zeichen pro Antwort |
| **Snippetfähigkeit** | Antworten müssen auch für sich allein stehen und Sinn ergeben. Kurzer zusammenfassender Satz am Anfang, tiefere Erklärung folgt |
| **Quellangaben** | FAQs können ausnahmsweise mit Quellangaben hinsichtlich E-E-A-T und AIO versehen werden |
| **CMS** | FAQs im CMS als FAQ auszeichnen (Haken anwählen) |

## I. SEO- & AIO-Checkliste

### Keyword-Integration

- [ ] HKW in H1 (ohne Stopwords, ohne Satzzeichen, ohne Flexion)
- [ ] HKW in Meta-Title ganz am Anfang (unveränderter Form)
- [ ] HKW in Meta-Description (Stopwords erlaubt)
- [ ] HKW im Intro so weit vorn wie möglich
- [ ] HKW in mind. jedem 2. H2 enthalten
- [ ] HKW im Text sowohl mit als auch ohne Stopwords verwendet
- [ ] Doppelpunkte im HKW zählen **nicht** als Keyword-Nutzung
- [ ] Keywords natürlich/organisch eingebunden, Lesefluss nicht gestört

### Verlinkung

- [ ] Interne Links zu verwandten /urlaub/-Seiten
- [ ] Interne Links zu /reisemagazin/-Artikeln
- [ ] Verlinkung auf Facettenseiten aus Zielgruppenabschnitten
- [ ] Verlinkung auf „Beste Reisezeit"-Seite (falls vorhanden)
- [ ] Teaserreihe mit Ort-Verlinkungen vorhanden

### Bilder

- [ ] Alt-Texte aller Bilder beschreibend, max. 125 Zeichen, mit Keyword
- [ ] Bilder einheitlich bearbeitet, in Bildreihen stimmig
- [ ] Text-Bild-Konsistenz geprüft (keine Text-Bild-Scheren)

### AIO-Optimierung

- [ ] FAQ-Sektion vorhanden (mind. 3 Fragen)
- [ ] Antworten snippetfähig (kurze Zusammenfassung am Anfang)
- [ ] Aussagen zitierfähig (präzise, als Quelle tauglich)
- [ ] Outro als Zusammenfassung vorhanden

### Allgemein

- [ ] Gesamtzeichenanzahl 11.000–15.000
- [ ] Keine Keyword-Kannibalisierung mit bestehenden Seiten geprüft
- [ ] Keine Dopplungen zum „Beste Reisezeit"-Content
- [ ] FAQs im CMS als FAQ ausgezeichnet
- [ ] „Gute Gründe" werden im weiteren Text erneut aufgegriffen

## J. Tonalitäts- und Stil-Guidelines

### Grundprinzipien

| Kategorie | Vorgabe |
|-----------|---------|
| **Textfunktion** | Inspirativ mit Ratgeber-Charakter; DERTOUR als Reiseexperte hervorheben |
| **Tonalität** | Hilfreich, professionell, konkret. Kein Werbeton, keine Floskeln |
| **Expertise** | „Wir bei DERTOUR", „die DERTOUR-Reiseexperten" o. Ä. |
| **Ansprache** | „du" in Maßen; bei Familien/Gruppen auch „ihr" |
| **Zahlen** | Bis zwölf ausschreiben, danach Ziffer. **Ausnahme:** Entfernungen und Zeitangaben immer als Ziffer |
| **Markenname** | DERTOUR stets in Großbuchstaben |

### Stil-Dos und -Don'ts

| So ja ✓ | So nicht ✗ |
|---------|------------|
| Aussagekräftige, präzise, lebhafte Formulierungen – Informationsgehalt im Vordergrund | Schwärmerische, übertrieben blumige oder umgangssprachliche Formulierungen („abchillen", „Wasserratten") |
| Aussagen begründen: „Der Aufstieg lohnt sich, **weil** …" | Unbegründete Behauptungen: „Bei einer Tour siehst du auf jeden Fall Giraffen" |
| Begriffe erklären, Zusammenhänge sichtbar machen | Fachbegriffe unerklärt stehen lassen |
| „Mit etwas Glück bekommst du Giraffen zu sehen" | Leistungsversprechen oder garantierte Erlebnisse |
| Roter Faden, logischer Aufbau, klare Struktur | Kurze Stakkato-Sätze oder lange verschachtelte Sätze |
| Destinationsspezifische, inspirative Headlines | Austauschbare Headlines wie „Sonne, Strand und Meer" oder „von Italien bis Frankreich" |

### Nicht erwünscht

| Kategorie | Details |
|-----------|---------|
| **Inhaltlich** | Belanglose Fakten (Landesgröße, detaillierte Ortsbeschreibungen, Einreiseformalitäten – sofern kein USP). Schwankende Informationen (genaue Preise, Namen von Angestellten) |
| **Stilistisch** | (KI)-Floskeln: generische Phrasen, Füllwörter, Synonymketten, Halbwissen ohne Belege, Übertreibungen („einer der spektakulärsten Strände weltweit"), Pseudo-freundliche Formulierungen („du wirst diese Reise lieben") |
| **Richtlinien** | No-Go-Wortliste, Tierschutzrichtlinien und Nachhaltigkeitsrichtlinien beachten (Links im Agentur-Briefing) |

## M. Seitenstruktur auf einen Blick

```
Meta-Title
Meta-Description
├── H1 + Subline + Keyvisual
├── Gute Gründe (3 Bullets)
├── Intro (300–500 Zeichen)
├── Schönste Orte (H2 + Subline + Teaserreihe)
├── [Optional] Schönste Strände (H2 + Subline)
├── Beliebte Sehenswürdigkeiten (H2 + Subline)
├── [Optional] Optionales Thema (H2 + Subline)
├── Zielgruppenabschnitt(e) (H2 + Subline + Experten-Tipp)
├── Klima (H2 + Subline + Infokasten + ggf. Tabelle)
├── FAQ (3–7 Fragen)
└── Outro (3–4 Sätze)
```
